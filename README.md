## Requirements

- Node version: 14.18.0
- NPM version: 6.14.15
- MongoDB

## Install dependencies

- npm install

## Start frontend project

- cd calendar-frontend
- yarn dev

## Start backend project

- cd calendar-backend
- npm run dev
